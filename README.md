# Brute-Force-Password-Simulation

Repositori ini berisi kode sumber dan dokumentasi untuk simulasi analisis kompleksitas waktu algoritma *Brute Force*.

## Deskripsi Proyek
Proyek ini bertujuan untuk menganalisis pertumbuhan waktu eksekusi (kompleksitas $O(k^n)$) pada algoritma *Brute Force* dalam pemecahan kata sandi sederhana. Simulasi dilakukan dengan menggunakan bahasa pemrograman Python.

## Struktur File
- `brute_force.py`: Kode sumber utama program simulasi.

## Cara Menjalankan
1. Pastikan Python 3.x telah terinstal di perangkat Anda.
2. Clone repositori ini atau download file `brute_force.py`.
3. Buka terminal pada folder tersebut.
4. Jalankan perintah berikut:
   ```bash
   python brute_force.py
